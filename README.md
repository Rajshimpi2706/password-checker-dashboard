**🔐 Password Strength Checker Dashboard**
A professional, modern, and interactive web dashboard to audit password security using Flask, Bootstrap, and cybersecurity best practices.
________________________________________
🚀 Features
•	✅ Real-time password strength analysis
•	✅ Breach detection using HaveIBeenPwned API
•	✅ Entropy and estimated crack time calculation
•	✅ Animated UI with 3D tilt, dark mode toggle, and confetti celebration
•	✅ Responsive, accessible, and mobile-ready design
•	✅ Fully customizable and open source

🧰 Installation
🔹 Clone the Repository
git clone https://github.com/Rajshimpi2706/password-checker-dashboard.git
cd password-checker-dashboard
🔹 Create a Virtual Environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
🔹 Install Dependencies
pip install -r requirements.txt
🔹 Run the Application
python app.py
Visit http://127.0.0.1:5000 in your browser.
________________________________________
🛡️ How It Works
•	Password is evaluated for length, characters, digits, symbols
•	Score & strength are calculated out of 10
•	Entropy & crack time are estimated
•	Breach check is performed using the HIBP API
•	Visual feedback and celebration animations show results dynamically
